# Security policy

## Supported versions

| Version | Supported |
| ------- | --------- |
| latest tagged release | yes |
| earlier tags          | no  |

## Reporting

Please report suspected vulnerabilities privately to:

  security@your-domain.example

Do not open a public issue.

Acknowledgement target: 7 days.
Triage target: 30 days.

## Scope

This repository is a bounded public artefact. The boundary of what is
in scope is defined by:

- `CLAIM_BOUNDARY.md` (per indexed repo) — what is claimed.
- `NON-CLAIMS.md`                         — what is not claimed.
- `PUBLIC_DISCLOSURE_BOUNDARY.md`         — what is disclosed publicly.
- `CHANGELOG.md`                          — release-discipline header.

Reports outside these documents may be closed as out of scope.
